Clinical RAG Decision Support Tool

RAG FAISS Llama 3 8B Code

Clinical decision support tool for querying peer-reviewed diabetes research via natural language, grounding every answer in evidence from three landmark trials (LABS2, LookAHEAD, GRADE) using a local LLaMA 3 8B model.

  • RAG pipeline over 80+ PDFs with FAISS retrieval, source diversity enforcement, and MEDIUM/HIGH/LOW confidence scoring via L2 distance.
  • Patient Profile Mode generates targeted multi-query evidence summaries for individual clinical characteristics.
  • 20-query automated test suite across answerable, edge case, and adversarial categories; runs fully local with no external API calls.

mortgage credit risk

Python XGBoost SHAP Code

End-to-end ML pipeline for predicting residential mortgage default risk (90+ days delinquency) using Fannie Mae's Single-Family Loan Performance dataset, directly extending my banking background into the US credit market.

  • Benchmarks logistic regression, XGBoost, and LightGBM on origination-level features (FICO, LTV, DTI, interest rate) with cross-validated AUC-ROC, KS statistic, and Brier score.
  • SHAP values for model interpretability — critical for regulatory transparency in credit risk applications.
  • Modular pipeline (ingest → features → model → evaluate) with Parquet-based storage and full calibration diagnostics.

Poisson & Football

React Statistics Simulation Live Code

Bilingual (EN/ES) web app that implements five Poisson family statistical models for football score prediction, computed client side, with match simulation, season tables, and model diagnostics.

  • Models span Maher (1982) to bivariate Poisson and negative binomial, including Dixon-Coles with time decay and low-score correction.
  • Data pipeline fetching historical CSVs from football-data.co.uk; parameters estimated via maximum likelihood with iterative scaling.
  • Originally a UNAM thesis project in R Shiny; rebuilt from scratch with React and Vite.

penalty kicks

Python Game Theory Reinforcement learning Code

Q-Learning agent trained on 482 real European penalty kicks to exploit goalkeeper behavioural bias, achieving 90% win rate vs a Nash equilibrium baseline of 50%, grounded in Chiappori et al. (2002).

  • Payoff matrix derived from real match data; agent learns an anti-copycat strategy across three shot directions with memory-dependent Q-values.
  • Curriculum training across four goalkeeper personalities (random, stochastic copycat, adversarial, statistical) to prevent overfitting to a single opponent style.
  • Interactive Streamlit shootout game with real-time pattern detection; 18-test suite covering environment, agent, and all keeper strategies.

football soccer

Sports analytics EDA Quarto Live Code

Exploratory analysis of 30+ years of English football across four league tiers, testing whether the Premier League has become more competitive over time through attack, defense, and refereeing indicators.

  • Documented the erosion of home advantage since 1993, away goals up ~30% since 2018, draws declining, away wins rising.
  • Cluster detection and PCA on match-level metrics to profile team styles; alluvial diagrams for tier promotion and relegation dynamics.
  • Referee bias analysis on card frequency by match context; published as an interactive Quarto book with seasonal visualizations.
